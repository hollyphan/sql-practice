# SQL Practice

Self-authored SQL problems and solutions, written and solved independently against the Parks & Recreation dataset (the practice dataset used in Alex the Analyst's SQL tutorial series). Problems are original, not copied from the tutorial's exercises or answer key. Progression runs beginner through advanced.

For original analytics project work, see [retail-ops-analytics](https://github.com/hollyphan/retail-ops-analytics).

## Structure

- `fundamentals/` — SELECT, WHERE, GROUP BY, HAVING, JOINs, aggregations, string/date functions
- `intermediate/` — subqueries, window functions, CTEs
- `advanced/` — multi-CTE queries, recursive CTEs, NTILE, PERCENT_RANK, LAG/LEAD, hierarchy traversal, salary gap analysis
- `challenges/` — LeetCode SQL 50

## Progress

- SQLBolt: all 18 lessons complete (SELECT through CREATE/ALTER/DROP TABLE)
- LeetCode SQL 50: complete across all sections (Select, Basic Aggregate Functions, Sorting and Grouping, Advanced Select and Joins, Subqueries)
- Parks & Rec dataset: self-authored problem sets complete across fundamentals, intermediate, and advanced tiers
